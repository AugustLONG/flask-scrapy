# flask-scrapy

利用 `Scrapy`来抓取数据，利用 `flask` 来进行展示

---

实现步骤：

* 创建 virtualenv.`pip install -r requirements.txt` 安装所需扩展

* `scrapy crawl`来抓取对应数据

* `python manage.py runserver`来执行
